What is research software?

it [research software] contains the research process: data is collected, read, preprocessed, analyzed, visualized and simulated using research software (LCRDM report)

Software is a form of data

  • can be stored, loaded, processed
  • used as input or output (of other software)
  • complex type of data
    • needs (more) context
    • has dependencies
    • requires maintenance
  • can only be interpreted in the context of what it does, not on its content alone (-> instructions)
  • it is usually dependent on other software (components)
  • requires maintenance, is dynamic
